Frequently Asked Questions

What is StemSlicer and who is it for?

StemSlicer is a desktop silence detection and stem export tool built for music producers, mixing engineers, and mastering engineers. If you receive stems with long silences between parts, StemSlicer automatically detects those gaps, chops the audio into clean clips, and exports a ready-to-open project file for your DAW. No more manually trimming silence from dozens of stems.

What DAWs are supported?

StemSlicer exports native project files for REAPER (.rpp), Pro Tools (AAF), Logic Pro (AAF), Cubase / Nuendo (AAF), Bitwig Studio (.dawproject), and Studio One (.dawproject). For FL Studio and Ableton Live, it exports one clean WAV per stem that you can drag directly into your session. Tracks are automatically color-coded in REAPER, Bitwig, and Studio One.

How is this different from "strip silence" in my DAW?

Built-in strip silence only works inside one DAW. StemSlicer works across DAWs — process your stems once and export project files for any DAW your collaborators use. It also handles batch processing of entire sessions, adds automatic track colors based on stem names (kicks = red, vocals = pink, etc.), and creates properly organized project files ready to mix. One tool, every DAW.

What audio file formats are supported?

StemSlicer accepts WAV, MP3, FLAC, OGG, AIF/AIFF, and M4A. You can drag and drop an entire folder of stems and it will process them all at once.

Does it work on macOS and Windows?

Yes. When you purchase StemSlicer, you get both the macOS and Windows versions. It runs natively — no browser, no Electron, no subscription, no internet connection required.
